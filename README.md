# Aalo Search
https://rakibgamecoc-hue.github.io/dummy-search/
Aalo Search is a lightweight, toy search engine project designed to demonstrate basic web search integration. By leveraging the Google Search API, the application provides a streamlined and efficient interface for users to perform quick queries. 

This repository houses the complete source code for the project, including the frontend user interface and the underlying backend logic required to process requests. Whether you are looking to understand how search API calls are structured or simply want to explore a functional search implementation, this project serves as a practical, easy-to-read reference point. 

Feel free to browse the repository, experiment with the provided code, and contribute to its development.
There is no  framework being used to create this project feel free to give any feedback as it would be very much appriciated.
